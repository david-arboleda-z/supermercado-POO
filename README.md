# Sistema de Supermercado en Java

Proyecto académico desarrollado en Java aplicando principios de Programación Orientada a Objetos (POO).

## Funcionalidades

- Gestión de productos
- Carrito de compras
- Cálculo de total de compra
- Simulación de compra en un supermercado

## Conceptos aplicados

- Clases y objetos
- Encapsulamiento
- Relaciones entre clases
- Diagramas UML

## Tecnologías

- Java
- UML
